# Loyal-Automotive-
Loyal Automotive is a web application designed to simplify bike service booking and spare parts management.

## features
- User authentication (Register/Login/Logout)
- Spare parts management 
- Cart & Wishlist
- Service booking system 
- Admin Dashboard
